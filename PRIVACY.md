# MacSentinel Privacy

MacSentinel is designed as a local-first Mac utility.

- System metrics stay on the Mac unless the user explicitly exports a JSON snapshot or Markdown report.
- The app does not include analytics, ads, tracking SDKs, or third-party network services.
- Exported reports can include process names, app names, storage paths, container names, and local host information.
- Cleanup actions move eligible cache contents to Trash instead of permanently deleting files.
- For the full-control edition, users may grant Full Disk Access to improve scan coverage.

For an App Store edition, add the final App Privacy answers in App Store Connect based on the exact feature set that remains in the sandboxed build.
